Quick guide

🔍 Search

Type anything — apartment (2BR-A), model number (23003), drawing number, sequence (10160), category, classification (e.g. Serviced) or tower. Tap a chip under the box to filter by tower / type / tier. Short forms work (SA, Res, HT) and spelling slips like "Servced" still find the right thing. Results are grouped by type, with a toolbar to switch between cards and a sortable list, copy the list or print/PDF them. In list view, tap any column header to sort. Press / to jump here, tap a number to copy it, star ★ to pin.

🏢 Federation

Tap a level to open the floor view, tap an apartment name to open its card. The Diagram button draws both towers as stacked tier bars.

📋 Data

All tables: click a header to sort, drag its edge to resize, use the Quick filter box to find rows. Drawings has the bulk bar: check rows → set status, date and/or category → Apply (each is optional). Paste list checks numbers from an email. Transmittal exports the checked rows as an issue sheet. Columns ▾ hides what you don't need. Apartments, drawings and models all have a Notes field.

✔ Health

Issues are grouped: 🔴 Errors, 🟡 Check these (spelling clashes, level gaps, unused models), 🟡 TIDP delivery, 🔵 Linking, ⚪ Info. Most rows have a one-tap fix. "Check these" are advisory — level gaps can be intentional (plant / void). Nothing is auto-deleted. Export issues to Excel for the team.

🧠 Map

Levels → Apartments → Models → Categories as strings. Tap a node to isolate its chain; numbers show floors / drawings per link. Simple view groups levels by tier.

💾 Import / Export

Import the project workbook once, import the TIDP, then Save Project (.json) — that one file restores everything offline. Choose a save folder so exports land where you want. For other projects use the custom mapping — it remembers your column choices. A red Danger zone at the bottom clears all data for a fresh project — it needs you to type DELETE, so it can't be hit by accident.

🔢 Naming rule

Set the numbering rule in Project info. Tokens: # = one digit, @ = one letter, * = anything within a segment. Example: 5013-KEO-*-*-*-INT-#####-T0. Health flags every drawing that breaks it.

↶ Safety & comfort

Undo reverses the last 6 changes. The browser warns before closing with unsaved edits, and you get a reminder every 10 changes. Nothing is ever deleted automatically. A− / A+ in the top bar change text size and Light / Dark is remembered next time. The Share button on search results copies a link that reopens the same search and filters.
