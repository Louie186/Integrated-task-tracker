Quick guide

🔍 Search

Type an apartment (2BR-A), a model number (23003), a drawing number or just its sequence (10160). Apartment and model searches open the same full card. Press / anywhere to jump here. Tap any number to copy it. Star ★ an apartment to pin it.

🏢 Federation

Tap a level to open the floor view, tap an apartment name to open its card. The Diagram button draws both towers as stacked tier bars.

📋 Data

All tables: click a header to sort, drag its edge to resize, use the Quick filter box to find rows. Drawings has the bulk bar: check rows → set status + date → Apply. Paste list checks numbers from an email. Transmittal exports the checked rows as an issue sheet. Columns ▾ hides what you don't need.

✔ Health

Issues are grouped: 🔴 Errors, 🟡 TIDP delivery, 🔵 Linking, ⚪ Info. Most rows have a one-tap fix. Duplicates are flagged only — never auto-deleted. Export issues to Excel for the team.

🧠 Map

Levels → Apartments → Models → Categories as strings. Tap a node to isolate its chain; numbers show floors / drawings per link. Simple view groups levels by tier.

💾 Import / Export

Import the project workbook once, import the TIDP, then Save Project (.json) — that one file restores everything offline. Choose a save folder so exports land where you want. For other projects use the custom mapping — it remembers your column choices.

🔢 Naming rule

Set the numbering rule in Project info. Tokens: # = one digit, @ = one letter, * = anything within a segment. Example: 5013-KEO-*-*-*-INT-#####-T0. Health flags every drawing that breaks it.

↶ Safety

Undo reverses the last 6 changes. The browser warns before closing with unsaved edits, and you get a reminder every 10 changes. Nothing is ever deleted automatically.
